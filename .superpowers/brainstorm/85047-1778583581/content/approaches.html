3 Approaches — Admin Redesign

Sidebar simplification + theme + product form. Pick the direction that feels right.

A

Grouped sidebar + Filament custom theme

Keep Filament as-is but group the 11 sidebar items into 3–4 navigation groups (Content, Customers, Automation, Settings). Add a warm gold/cream brand theme to replace the generic dark. Product form gets tabs. Fast to implement — no structural changes to Filament.

Pros

  • Minimal code changes
  • Stays within Filament conventions
  • Can ship in a day

Cons

  • Still 11 items (just grouped)
  • Filament theme customisation is limited
B

Grouped sidebar + full brand theme (recommended)

Same grouping as A, but use a custom Filament panel theme with your brand colours (warm white background, gold accents, clean typography). Hide low-traffic items (Config Referral, Email Triggers) behind a "Settings" group collapsed by default. Product form tabs. This is the sweet spot between effort and result.

Pros

  • Feels like a product built for Exclusive Doors
  • Much cleaner visual hierarchy
  • Most important items front and centre

Cons

  • ~2–3 days of work
  • Custom theme needs testing across all pages
C

Radically simplified — hide advanced features

Cut the sidebar to 6 items max: Dashboard, Produse, Clienți, Lead-uri, Promoții, and a single "Setări" page that contains categories, CMS, referral config, email templates, and triggers all in one settings screen. Product form tabs. Much steeper refactor — some Filament resources get merged.

Pros

  • Cleanest possible sidebar
  • Beginner-friendly for staff

Cons

  • Most complex to build
  • Merged settings page can itself become overwhelming
  • Harder to extend later
